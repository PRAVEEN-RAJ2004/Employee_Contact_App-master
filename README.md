EMPLOYEE CONTACT MANAGER APP --732923ITR046
Problem Statement
In many organizations, managing employee contact details manually is difficult and time-consuming. Important information like employee name, ID, phone number, and department may not be easily accessible when needed.
The problem is to design and develop an Android application called “Employee Contact Manager App” that allows users to:
•	Enter employee details
•	Store and display employee information
•	Perform telephony actions like calling, sending SMS, and sending email
•	Show confirmation dialog before performing actions
•	Navigate between screens using Intent
The application should be simple, user-friendly, and interactive.
OBJECTIVES
1.	To create an Android application using Java.
2.	To collect employee details through EditText fields.
3.	To use Spinner for department selection.
4.	To display employee details using ListView with CardView.
5.	To implement Intent for navigation between activities.
6.	To implement telephony features (Call, SMS, Email).
7.	To use AlertDialog for confirmation before actions.

Concept used 
Concept	Purpose
XML Layout	Used to design the user interface of the app screens.
EditText	Allows user to enter employee details like name, ID, phone number.
Spinner	Lets user select department from a dropdown list.
Button	Performs actions like Save, View, Call, SMS, and Email.
ListView	Displays employee details in a scrollable list format.
CardView	Shows each employee’s details in an attractive card style.
Intent	Used to navigate between activities and perform actions.
AlertDialog	Displays confirmation dialog before performing call or message.
